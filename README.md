# Week6 ゼミ

## WORK1 Git

### コミット編

- git init ：git の初期化
- git status ：現状を確認
- git add ：コミットする対象を選ぶ
- git commit ：add で選んだ対象をコミットする
  - コミット ：Git に登録すること

### ブランチ編

- git branch ~ ：~ ブランチを作成する
  - git branch ：ブランチ一覧を表示
- git checkout ~：~ ブランチに移動する
- git merge ~ ：~ ブランチを今いるブランチにマージする
